# Ideals in Algebraic Rings of Integers


## Constructing ideals

- [`all_ideals`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/algebraic_number_field/struct.OrderIdealsStructure.html#method.all_ideals) generate all ideals
- [`all_ideals_norm_eq`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/algebraic_number_field/struct.OrderIdealsStructure.html#method.all_ideals_norm_eq) generate all ideals of norm equal to n
- [`all_nonzero_ideals`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/algebraic_number_field/struct.OrderIdealsStructure.html#method.all_nonzero_ideals) generate all non-zero ideals
- [`all_nonzero_ideals_norm_le`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/algebraic_number_field/struct.OrderIdealsStructure.html#method.all_nonzero_ideals_norm_le) generate all non-zero ideals of norm at most n
- [`generated_ideal`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/structure/trait.IdealsArithmeticSignature.html#method.generated_ideal) The ideal generated by a list of elements
- [`principal_ideal`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/structure/trait.IdealsArithmeticSignature.html#tymethod.principal_ideal) The principal ideal generated by a
- [`unit_ideal`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/structure/trait.IdealsArithmeticSignature.html#method.unit_ideal)
- [`zero_ideal`](https://docs.rs/algebraeon-rings/latest/algebraeon_rings/structure/trait.IdealsArithmeticSignature.html#method.zero_ideal)

## Factoring example

```rust
use algebraeon::rings::algebraic_number_field::AlgebraicNumberFieldSignature;
use algebraeon::rings::algebraic_number_field::FullRankSublatticeWithBasisSignature;
use algebraeon::rings::polynomial::PolynomialFromStr;
use algebraeon::sets::structure::InjectiveFunction;
use algebraeon::{
    nzq::*,
    rings::{polynomial::Polynomial, structure::*},
};

// Construct the ring of integers Z[i]
let anf = Polynomial::<Rational>::from_str("x^2+1", "x")
    .unwrap()
    .algebraic_number_field()
    .unwrap();
let roi = anf.ring_of_integers();

// The ideal (27i - 9) in Z[i]
let ideal = roi.ideals().principal_ideal(
    &roi.outbound_order_to_anf_inclusion()
        .try_preimage(&Polynomial::from_str("27*x-9", "x").unwrap())
        .unwrap(),
);

// Factor the ideal
for (prime_ideal, power) in roi
    .ideals()
    .factorizations()
    .into_powers(roi.ideals().factor_ideal(&ideal).unwrap())
{
    println!("power = {power} prime_ideal_factor = {:?}", prime_ideal);
}

// There's not yet a nice way to print ideals so the output is messy
// But it prints the following factorization into primes
// ideal = (1+i) * (1+2i) * (3)^2
```
